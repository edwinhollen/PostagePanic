Postage Panic

For modern browsers only.

How to play

Observe the parcel's destination city and state. Accept correct parcels; deny incorrect parcels.

Description

Improve your knowledge of U.S. state capitals with this tiny game written in Javascript for the HTML5 canvas! Also uses the ChanceJS library.

Under the hood

  • Silky smooth 60FPS thanks to modern browsers and their respective implementation of the requestAnimationFrame API.
  • Some browsers may render a blurry representation of the game; this is due to a lack of native support for the imageSmoothingEnabled canvas context property.
  • Open the developer console for cheats mode.
